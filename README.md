# ResearchGroup

## Installing Library(for Arduino Library Manager)

+ Keypad
+ MFRC522

## Connect

Pin layout should be as follows:

| RFID | Arduino |
|:-----|--------:|
| 3.3V | 3.3V    |
| RST  | 9       |
| GND  | GND     |
| MISO | 12      |
| MOSI | 11      |
| SCK  | 13      |
| SDA  | 10      |
| IRQ  | -       |

 **IRQ** Not connected
